1) Установить Denwer - прогу что нам давал Шаров. Тут будет сервер твоей БД.
2) Создать там пустую базу данных с именем nochnoiklubbd
3) Импортировать таблицы из nochnoiclubbd.sql файла (лежит в корне каталога проекта).
4) Установить node.js версии 5.0.0 (скачать с оф. сайта node.js) нужной разрядности (х86/х64).
5) Запустить коммандную строку Windows в папке WebSQLServiceKulik.
6) Теперь можно запускать веб-сервер (наш-сайт) с помощью комманды: node main.js в ком. строке Windows.
7) Сервер запущен. Если денвер не вырубать файлом Stop Denwer.exe, сайт будет доступен по адресу - localhost:8080 в любом браузере. Советую Mozila FireFox ESR версию - крутой браузер и четко работает, красиво рисует элементы сайта.
8) Если есть ноут - будет идеально. Надо будет учителю на своем ноуте показать, либо просить кого-то у кого ноут есть (Портос, например)- загрузить твою БД на его ноут и скинуть ему папку с проектом.
